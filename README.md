# Task Manager

A Full Stack Task Manager application built using React, Flask, MySQL, and Docker.

## Features

- Add Tasks
- Update Tasks
- Delete Tasks
- View All Tasks
- MySQL Database
- REST API using Flask
- React Frontend
- Dockerized Application

## Tech Stack

- React
- Flask
- MySQL
- Docker
- Docker Compose

## Project Structure

```
TaskManager
│
├── backend
├── frontend
├── docker-compose.yml
└── README.md
```

## Run using Docker

```bash
docker compose up --build
```

Frontend:
http://localhost:3000

Backend:
http://localhost:5000/tasks

Database:
MySQL (Port 3307)
